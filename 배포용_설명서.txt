🎯 고성현 때리기 게임 배포용

📁 파일 구성:
- index.html (메인 게임 파일)
- style.css (스타일 파일)
- script.js (게임 로직 파일)
- ball.jpg (과녁 이미지)
- README.md (게임 설명서)
- 배포용_설명서.txt (이 파일)

🚀 실행 방법:
1. 모든 파일을 같은 폴더에 저장
2. index.html 파일을 웹 브라우저로 열기
3. 게임 시작!

🎮 게임 특징:
- 5가지 총기 (권총, 샷건, AK-47, 스나이퍼, 바주카포)
- 총알 30발
- 움직이는 과녁
- 피 효과와 폭발 효과
- 재미있는 말풍선

🎯 조작법:
- 마우스: 과녁 클릭으로 발사
- 키보드: 1-5로 무기 선택, R로 재장전, 스페이스바로 재시작

📦 배포 방법:
1. 모든 파일을 ZIP으로 압축
2. "고성현때리기.zip" 파일명으로 저장
3. 다른 사람에게 전송

🌐 웹 배포 (선택사항):
- GitHub Pages, Netlify, Vercel 등에 업로드
- URL만 공유하면 바로 플레이 가능

즐거운 게임 되세요! 🎮
